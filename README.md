## 팀원
- 기획/디자인 : 박다비다</br>
- 프론트엔드 : 이소윤, 이지현, 전소정</br>
- 벡엔드 : 김민희, 양성원, 윤수윤</br>


## 주요 기능
- 회원가입, 로그인
- 일기 작성
- 일기 키워드 검색
- 나만의 감정 통계 그래프 확인
- 365 Q&A
